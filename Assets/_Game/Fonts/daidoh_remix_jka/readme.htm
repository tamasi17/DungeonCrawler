■　は　じ　め　に　■

　「自分もオリジナルの欧文フォントを作ろう！」と思い立ったのは、EmigreやT-26、FUSE FONTなどを知ってからで、「欧文は文字数が少ないからカンタン？」と思って作り始めましたが、細かいルールがあって意外に大変でした。
　最近ではオリジナルフォントをWeb上で配布する人も増え、ささやかながらも、このサイトを訪れダウンロードしてくれる人も増えました。
　これからも余りペースは速くないと思いますが、徐々に新作をリリースして生きたいと思っています。まずは、以下のフォントをお試し下さい。一部を除き、フリーウェアとなっております。詳しくはこのRead Meファイルをお読み下さい。また、各フォント名は、その名に由来したアーティストへのオマージュとして、付けられています。

■　イ　ン　ス　ト　ー　ル　方　法　■

●Macintosh
Type1 Fontの場合は"スーツケース"ファイルと"プリンタフォント"ファイルの両方を、Truetypeの場合は"スーツケース"ファイルを『システムフォルダ』の中の『フォントフォルダ』にコピーしてご使用下さい。

●Windows
１）Windowsコントロールパネルのアイコンを開く。
２）『Fonts』パネルのアイコンを開き、Fontsパネルを表示させる。
３）『Add...』ボタンをクリックし『Add Fonts』パネルを表示させる。
４）『Drives』リストと『Directories』リストを使ってインストールするフォントの含まれているディレクトリに移動する。
５）『List of Fonts』からインストールしたいフォントの名前を選択する。
６）『OK』ボタンをクリックする。

#Installation.

@Macintosh
Copy a "suitcase" file on "Font folder" in "System folder" in the case of Truetype, and use both of the "suitcase" file and the "printer font" file in the case of Type1Font.

@Windows
1) Open the icon of the Windows control panel.
2) "Fonts" Open the icon of the panel, and make it indicate a Fonts panel.
3) It clicks on the "Add..." button, and make it indicate a "Add Fonts" panel.
4) "Drives" It is moved by using the list and the "Directories" list to the directory which a font to install is contained to.
5) Choose the name of the font which it wants to install from "List of Fonts".
6) It clicks on the "OK" button.
